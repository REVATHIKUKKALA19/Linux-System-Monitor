# Linux System Monitor

A simple Python-based system monitoring tool that displays system information and saves it to a log file.

## Features

- CPU Usage
- Memory Usage
- Disk Usage
- Running Processes
- Log File Generation

## Technologies Used

- Python
- psutil

## Installation

```bash
pip install psutil
```

## Run

```bash
python monitor.py
```

## Output

- Displays CPU Usage
- Displays Memory Usage
- Displays Disk Usage
- Displays Running Processes
- Saves report to `system.log`
