# Linux System Monitor

A simple Python-based system monitoring tool.

## Features

- CPU Usage
- Memory Usage
- Disk Usage
- Running Processes
- Save output to system.log

## Technologies Used

- Python
- psutil

## Installation

```bash
pip install psutil
```

## Run

```bash
python monitor.py
```

## Output

- Displays CPU, Memory, and Disk Usage
- Lists Running Processes
- Saves report to system.log
